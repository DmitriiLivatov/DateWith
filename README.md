# Interactive Date Invitation

An interactive web-based invitation system built with HTML, CSS, and JavaScript.

## Features
* **Envelope Animation:** Realistic opening sequence using CSS animations and clip-path.
* **Dynamic Button Logic:** The "No" button utilizes randomized coordinate shifts on hover to prevent selection.
* **Step-by-Step Selection:** Sequential screens for date, time, and location selection.
* **Telegram Integration:** Data transmission via Telegram Bot API for real-time response notifications.
* **Lottie Animations:** Vector-based icons and graphics rendered via the Lottie-web library.

## Technologies Used
* HTML5 / CSS3
* Vanilla JavaScript
* Lottie Framework (Vector Animation)
* Flatpickr (Datetime Picker)
* Telegram Bot API (Data Processing)

## Deployment Instructions
1. Clone the repository to your local machine.
2. Navigate to `script.js` and input your specific `TELEGRAM_BOT_TOKEN` and `TELEGRAM_CHAT_ID`.
3. Ensure all `.json` animation files are located in the root directory.
4. Deploy the project via GitHub Pages or any static hosting provider.
